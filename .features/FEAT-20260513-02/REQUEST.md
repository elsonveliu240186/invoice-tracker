# Invoice Export — PDF generation & email delivery to clients

## Raw user request

Enable invoice PDF generation and the ability to send it directly to clients.

## Detailed requirements

1. **PDF generation** — generate a professional, print-ready PDF from an existing invoice (all line items, totals, company/client details, logo if available).
2. **Send to client by email** — from the invoice detail view, trigger an email to the client's registered email address with the PDF attached.
3. The email should have a sensible subject ("Invoice #XXXX from [Company]") and a short body.
4. Delivery status should be visible in the UI (e.g. "Sent on DD/MM/YYYY").
