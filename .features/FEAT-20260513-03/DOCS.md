# DOCS — FEAT-20260513-03: Invoice Sharing (DOCX template rendering, PDF conversion, email delivery)

Generated by the documentation agent on 2026-05-14.

## Summary

This feature adds DOCX-template-based invoice rendering (poi-tl), LibreOffice headless PDF conversion, template upload/management, and email delivery. Documentation was hand-constructed because the backend requires a running Postgres instance that is not available in the documentation agent environment. All endpoint paths were verified against the controller source files.

**Important path deviation from PLAN.md**: `InvoiceRenderController` exposes `/docx-pdf` and `/docx-email` sub-paths (not `/pdf` and `/send-email` as the plan specified) to avoid request mapping conflicts with `InvoiceController` which already owns those paths from FEAT-02. See ADR-016 in `docs/ARCHITECTURE.md`.
