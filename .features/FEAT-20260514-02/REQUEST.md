# invoice-template-editor-and-full-lifecycle

## Raw user request

Currently the invoice is uploaded but it is not possible to manage the invoice by personalizing the template in the invoices menu.

We need to add:

1. **Template editor** — ability to edit invoice templates directly from the Invoices menu (not just in Settings). Users should be able to personalize the template.
2. **Save invoices as PDF or DOCX** — generate and save the document once the invoice is ready.
3. **Download** — users can download the generated PDF or DOCX from the invoice detail page.
4. **Preview** — users can preview the invoice (PDF or rendered HTML) inline before sending/downloading.
5. **Send per email** — send the invoice to the customer directly from the invoice detail page.

The goal is a complete invoice lifecycle: create → edit template → preview → generate (PDF/DOCX) → download or send by email.
