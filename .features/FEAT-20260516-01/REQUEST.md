# Expense Tracking with Category Dashboard

## Raw user request

Create a new feature for expenses tracking. Requirements:

- A dedicated Expenses page with a dashboard at the top as the main view
- Dashboard shows a breakdown of expenses per category (amounts, counts)
- Ability to add expenses with: date, value (amount), category
- Use default popular app categories with icons to represent them (e.g. Food & Drink, Transport, Housing, Health, Entertainment, Shopping, Travel, Education, Utilities, Other)
- After the dashboard show the list of individual expenses
- The design should follow the same standards as the rest of the app (modern SaaS, icon actions, same button patterns as invoices/clients)
