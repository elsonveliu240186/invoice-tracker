# Expense Dashboard Charts — By Month and By Category

User request:

Add expense graphs to the dashboard, similar to the existing revenue-by-month and invoice-status charts. Two new charts are required:

1. **Expense by Month** — a bar or line chart showing total expenses per calendar month (same style as the existing revenue-by-month chart).
2. **Expense by Category** — a pie or donut chart showing expense totals broken down by category (same style as the existing invoice-status chart).

Both charts should use real data from the expenses API and respect the same date-range filter already in place on the dashboard.
