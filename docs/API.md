# API reference

Regenerated by the **documentation** subagent from the live `/v3/api-docs` JSON each feature. Do not hand-edit between feature runs — your changes will be overwritten.

To regenerate manually:

```powershell
# from project root
cd backend
./mvnw spring-boot:run    # background
curl http://localhost:8080/v3/api-docs > ../docs/openapi.json
```
